# Mapa de la organización territorial del Imperio Romano en el año 395

Este mapa web muestra la organización territorial en provincias del Imperio Romano en el año 395, fecha de la muerte del emperador Teodosio, y última vez que el Imperio Romano formó una única unidad en torno al Mar Mediterráneo.

El mapa web se ha hecho con la versión 4.2 de JavaScript para ArcGIS.
